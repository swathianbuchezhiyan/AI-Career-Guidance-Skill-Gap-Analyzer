# AI Career Guidance and Skill Gap Analyzer

## Project Overview
This project recommends suitable career paths based on user skills using Python and Pandas. It analyzes skill similarity and identifies missing skills for improvement.

---

## Features
- Skill matching system  
- Top 3 career suggestions  
- Skill gap analysis  

---

## Technologies Used
- Python  
- Pandas  

---

## How It Works
1. User inputs skills  
2. System compares skills with job roles  
3. Calculates match score  
4. Identifies missing skills  
5. Suggests top career options  

---

## Output Example
- Data Scientist → 75 percent match  
- Data Analyst → 80 percent match  
- Machine Learning Engineer → 50 percent match

- Skill matching algorithm
